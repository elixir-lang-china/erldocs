<h1>ct_ftp</h1> <h2 class="modsummary">FTP client module (based on the FTP support of the INETS application).</h2> <div class="description"> <p>FTP client module (based on the FTP support of the INETS application). </p></div> <div class="section"><h4>DATA TYPES</h4><span id="types"> </span> <taglist> <dt><code>connection() = handle() | target_name() (see module ct)</code></dt> <item><span id="type-connection"> </span> </item> <dt><code>handle() = handle() (see module ct_gen_conn)</code></dt> <item><span id="type-handle"> </span> <p>Handle for a specific ftp connection.</p> </item> </taglist></div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="put/3">put(KeyOrName, LocalFile, RemoteFile) -&gt; {ok, Handle} | {error, Reason}</h3> <ul class="type"> <li><code>KeyOrName = Key | Name</code></li><li><code>Key = atom()</code></li><li><code>Name = target_name() (see module ct)</code></li><li><code>Handle = handle()</code></li></ul> <div class="description"><span id="open-1"> </span> <p>Open an FTP connection to the specified node. </p><p>You can open one connection for a particular <code>Name</code> and use the same name as reference for all subsequent operations.
