Channel</h3> <ul class="type"> <li><code>Channel = #objref</code></li> </ul> <div class="description"> <p>This operation creates a new instance of a <a href="CosEventChannelAdmin_EventChannel.html" class="seealso">Event Channel</a> using the default settings.</p> </div> </div> <div class="function"> <h3 id="start_channel/1">start_channel(Options) -&gt; Channel</h3> <ul class="type"> <li><code>Options = [Option]</code></li> <li><code>Option = {pull_interval, Seconds} | {typecheck, Boolean} | {maxEvents, Integer} | {blocking, Boolean}</code></li> <li><code>Channel = #objref</code></li> </ul> <div class="description"> <p>This operation creates a new instance of a <a href="CosEventChannelAdmin_EventChannel.html" class="seealso">Event Channel</a></p> <p /> <list type="bulleted"> <item><code>{pull_interval, Seconds}</code> - determine how often Proxy Pull Consumers will check for new events with the client application. The default value is 20 seconds.</item> <item><code>{typecheck, Boolean}</code> - if this option is set to true the proxies will check if the supplied client object is of correct type. The default value is false.</item> <item><code>{maxEvents, Integer}</code> - this option determine how many events the <code>ProxyPullSuppliers</code> will store before discarding events. If the limit is reached events will be discarded in any order. The default value is 300.</item> <item><code>{blocking, Boolean}</code> - this option determine the behavior of the channel when handling events internally. If set to <code>true</code> the risk of a single event supplier floods the system is reduced, but the performance may also be reduced. The default value is <code>true</code>.</item> </list> </div> </div> <div class="function"> <h3 id="start_channel_link/0">start_channel_link() -&gt; Channel</h3> <ul class="type"> <li><code>Channel = #objref</code></li> </ul> <div class="description"> <p>This operation creates a new instance of a <a href="CosEventChannelAdmin_EventChannel.html" class="seealso">Event Channel</a>, which is linked to the invoking process, using the default settings.</p> </div> </div> <div class="function"> <h3 id="start_channel_link/1">start_channel_link(Options) -&gt; Channel</h3> <ul class="type"> <li><code>Options = [Option]</code></li> <li><code>Option = {pull_interval, Seconds} | {typecheck, Boolean} | {maxEvents, Integer} | {blocking, Boolean}</code></li> <li><code>Channel = #objref</code></li> </ul> <div class="description"> <p>This operation creates a new instance of a <a href="CosEventChannelAdmin_EventChannel.html" class="seealso">Event Channel</a>, which is linked to the invoking process, with settings defined by the given options.
