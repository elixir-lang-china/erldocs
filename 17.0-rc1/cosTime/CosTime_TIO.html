<h1>CosTime_TIO</h1> <h2 class="modsummary">This module implements the OMG CosTime::TIO interface.</h2> <div class="description"> <p>To get access to the record definitions for the structures use: <br /> <code>-include_lib("cosTime/include/*.hrl").</code></p> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="'_get_time_interval'/1">'_get_time_interval'(TIO) -&gt; TimeInterval</h3> <ul class="type"> <li><code>TIO = #objref</code></li> <li><code>TimeInterval = #'TimeBase_IntervalT{lower_bound, upper_bound}</code></li> <li><code>lower_bound = upper_bound = ulonglong</code></li> </ul> <div class="description"> <p>This operation returns the interval associated with the target object.</p> </div> </div> <div class="function"> <h3 id="spans/2">spans(TIO, UTO) -&gt; Reply</h3> <ul class="type"> <li><code>TIO = UTO = OtherTIO = #objref</code></li> <li><code>Reply = {OverlapType, OtherTIO}</code></li> <li><code>OverlapType = 'OTContainer' | 'OTContained' | 'OTOverlap' | 'OTNoOverlap'</code></li> </ul> <div class="description"> <p>This operation returns a <em>OverlapType</em> depending on how the interval in the target object and the timerange represented by the UTO object overlap. If the OverlapType is 'OTNoOverlap' the out parameter represents the gap between the two intervals. If OverlapType is one of the others, the out parameter represents the overlap interval. The definitions of the OverlapType's are:</p> <p /> <list type="bulleted"> <item>'OTContainer' - target objects lower and upper limits are, respectively, less or equal to and greater or equal to given object's.</item> <item>'OTContained' - target objects lower and upper limits are, respectively, greater or equal to and less or equal to given object's.</item> <item>'OTOverlap' - target objects interval overlap given object's.</item> <item>'OTNoOverlap' - target objects interval do not overlap given object's.</item> </list> </div> </div> <div class="function"> <h3 id="overlaps/2">overlaps(TIO, OtherTIO) -&gt; Reply</h3> <ul class="type"> <li><code>TIO = OtherTIO = AnotherTIO = #objref</code></li> <li><code>Reply = {OverlapType, AnotherTIO}</code></li> <li><code>OverlapType = 'OTContainer' | 'OTContained' | 'OTOverlap' | 'OTNoOverlap'</code></li> </ul> <div class="description"> <p>This operation returns a <em>OverlapType</em> depending on how the interval in the target object and the timerange represented by the TIO object overlap. The OverlapType's are described under spans/2.</p> </div> </div> <div class="function"> <h3 id="time/1">time(TIO) -&gt; UTO</h3> <ul class="type"> <li><code>TIO = UTO = #objref</code></li> </ul> <div class="description"> <p>This operation returns a UTO in which the interval equals the time interval in the target object and time value is the midpoint of the interval.</p> </div> </div> </div>
