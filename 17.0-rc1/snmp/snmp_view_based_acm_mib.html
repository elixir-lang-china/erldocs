<h1>snmp_view_based_acm_mib</h1> <h2 class="modsummary">Instrumentation Functions for SNMP-VIEW-BASED-ACM-MIB</h2> <div class="description"> <p>The module <code>snmp_view_based_acm_mib</code> implements the instrumentation functions for the SNMP-VIEW-BASED-ACM-MIB, and functions for configuring the database. </p> <p>The configuration files are described in the SNMP User's Manual.</p> <span id="configure"> </span> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="configure/1">configure(ConfDir) -&gt; void()</h3> <ul class="type"> <li><code>ConfDir = string()</code></li> </ul> <div class="description"> <p>This function is called from the supervisor at system start-up. </p> <p>Inserts all data in the configuration files into the database and destroys all old rows with StorageType <code>volatile</code>. The rows created from the configuration file will have StorageType <code>nonVolatile</code>. </p> <p>All <code>snmp</code> counters are set to zero. </p> <p>If an error is found in the configuration file, it is reported using the function <code>config_err/2</code> of the error report module, and the function fails with the reason <code>configuration_error</code>. </p> <p><code>ConfDir</code> is a string which points to the directory where the configuration files are found. </p> <p>The configuration file read is: <code>vacm.conf</code>. </p> <span id="reconfigure"> </span> </div> </div> <div class="function"> <h3 id="reconfigure/1">reconfigure(ConfDir) -&gt; void()</h3> <ul class="type"> <li><code>ConfDir = string()</code></li> </ul> <div class="description"> <p>Inserts all data in the configuration files into the database and destroys all old data, including the rows with StorageType <code>nonVolatile</code>. The rows created from the configuration file will have StorageType <code>nonVolatile</code>. </p> <p>Thus, the data in the SNMP-VIEW-BASED-ACM-MIB, after this function has been called, is the data from the configuration files. </p> <p>All <code>snmp</code> counters are set to zero. </p> <p>If an error is found in the configuration file, it is reported using the function <a href="snmpa_error.html#config_err" class="seealso">config_err/2</a> of the error report module, and the function fails with the reason <code>configuration_error</code>. </p> <p><code>ConfDir</code> is a string which points to the directory where the configuration files are found. </p> <p>The configuration file read is: <code>vacm.conf</code>. </p> <span id="add_sec2group"> </span> </div> </div> <div class="function"> <h3 id="add_sec2group/3">add_sec2group(SecModel, SecName, GroupName) -&gt; Ret</h3> <ul class="type"> <li><code>SecModel = v1 | v2c | usm</code></li> <li><code>SecName = string()</code></li> <li><code>GroupName = string()</code></li> <li><code>Ret = {ok, Key} | {error, Reason}</code></li> <li><code>Key = term()</code></li> <li><code>Reason = term()</code></li> </ul> <div class="description"> <p>Adds a security to group definition to the agent config.
