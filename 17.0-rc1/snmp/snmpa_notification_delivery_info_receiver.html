<h1>snmpa_notification_delivery_info_receiver</h1> <h2 class="modsummary"> Behaviour module for the SNMP agent notification delivery information receiver. </h2> <div class="description"> <p>This module defines the behaviour of the notification delivery information receiver. </p> <p>A <code>snmpa_notification_delivery_info_receiver</code> compliant module must export the following functions: </p> <list type="bulleted"> <item> <p><a href="#delivery_targets" class="seealso">delivery_targets/3</a></p> </item> <item> <p><a href="#delivery_info" class="seealso">delivery_info/4</a></p> </item> </list> <p>The semantics of them and their exact signatures are explained below. </p> </div> <div class="section"> <h4>DATA TYPES</h4> <pre class="sh_erlang"> address() = A 4-tuple </pre> <span id="delivery_targets"> </span> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="delivery_targets/3">delivery_targets(Tag, Targets, Extra) -&gt; void()</h3> <ul class="type"> <li><code>Tag = term()</code></li> <li><code>Targets = [target()]</code></li> <li><code>target() = {Address, Port}</code></li> <li><code>Address = address()</code></li> <li><code>Port = integer()</code></li> <li><code>Extra = term()</code></li> </ul> <div class="description"> <p>Inform about target addresses. </p> <p>This is the first function called when a notification delivery is in progress. It informs the <code>receiver</code> which targets will get the notification. The result of the delivery will be provided via successive calls to <code>delivery_info/4</code> function, see below.</p> <span id="delivery_info"> </span> </div> </div> <div class="function"> <h3 id="delivery_info/4">delivery_info(Tag, Target, DeliveryResult, Extra) -&gt; void()</h3> <ul class="type"> <li><code>Tag = term()</code></li> <li><code>Target = target()</code></li> <li><code>target() = {Address, Port}</code></li> <li><code>Address = address()</code></li> <li><code>Port = integer()</code></li> <li><code>DeliveryResult = delivery_result()</code></li> <li><code>delivery_result() = no_response | got_response</code></li> <li><code>Extra = term()</code></li> </ul> <div class="description"> <p>Inform about delivery result.</p> <p>This function is called for each target in the <code>Targets</code> argument of the <code>delivery_targets/3</code> function, see above. </p> <p>The purpose is to inform the <code>receiver</code> of the result of the delivery (was the notification acknowledged or not) for each target.</p> </div> </div> </div>
