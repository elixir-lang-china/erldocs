<h1>snmp_standard_mib</h1> <h2 class="modsummary">Instrumentation Functions for STANDARD-MIB and SNMPv2-MIB</h2> <div class="description"> <p>The module <code>snmp_standard_mib</code> implements the instrumentation functions for the STANDARD-MIB and SNMPv2-MIB, and functions for configuring the database. </p> <p>The configuration files are described in the SNMP User's Manual.</p> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="configure/1">configure(ConfDir) -&gt; void()</h3> <ul class="type"> <li><code>ConfDir = string()</code></li> </ul> <div class="description"> <p>This function is called from the supervisor at system start-up. </p> <p>Inserts all data in the configuration files into the database and destroys all old rows with StorageType <code>volatile</code>. The rows created from the configuration file will have StorageType <code>nonVolatile</code>. </p> <p>All <code>snmp</code> counters are set to zero. </p> <p>If an error is found in the configuration file, it is reported using the function <code>config_err/2</code> of the error report module, and the function fails with the reason <code>configuration_error</code>. </p> <p><code>ConfDir</code> is a string which points to the directory where the configuration files are found. </p> <p>The configuration file read is: <code>standard.conf</code>. </p> </div> </div> <div class="function"> <h3 id="inc/1">inc(Name) -&gt; void()</h3> <h3 id="inc/2">inc(Name, N) -&gt; void()</h3> <ul class="type"> <li><code>Name = atom()</code></li> <li><code>N = integer()</code></li> </ul> <div class="description"> <p>Increments a variable in the MIB with <code>N</code>, or one if <code>N</code> is not specified.</p> </div> </div> <div class="function"> <h3 id="reconfigure/1">reconfigure(ConfDir) -&gt; void()</h3> <ul class="type"> <li><code>ConfDir = string()</code></li> </ul> <div class="description"> <p>Inserts all data in the configuration files into the database and destroys all old data, including the rows with StorageType <code>nonVolatile</code>. The rows created from the configuration file will have StorageType <code>nonVolatile</code>. </p> <p>Thus, the data in the SNMP-STANDARD-MIB and SNMPv2-MIB, after this function has been called, is from the configuration files. </p> <p>All <code>snmp</code> counters are set to zero. </p> <p>If an error is found in the configuration file, it is reported using the function <code>config_err/2</code> of the error report module, and the function fails with the reason <code>configuration_error</code>. </p> <p><code>ConfDir</code> is a string which points to the directory where the configuration files are found. </p> <p>The configuration file read is: <code>standard.conf</code>. </p> </div> </div> <div class="function"> <h3 id="reset/0">reset() -&gt; void()</h3> <div class="description"> <p>Resets all <code>snmp</code> counters to 0.</p> </div> </div> <div class="function"> <h3 id="sys_up_time/0">sys_up_time() -&gt; Time</h3> <ul class="type"> <li><code>Time = int()</code></li> </ul> <div class="description"> <p>Gets the system up time in hundredth of a second.</p> </div> </div> </div>
