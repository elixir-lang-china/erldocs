<h1>snmpa_supervisor</h1> <h2 class="modsummary">A supervisor for the SNMP agent Processes</h2> <div class="description"> <p>This is the top supervisor for the agent part of the SNMP application. There is always one supervisor at each node with an SNMP agent (master agent or sub-agent). </p> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="start_sub_sup/1">start_sub_sup(Opts) -&gt; {ok, pid()} | {error, {already_started, pid()}} | {error, Reason}</h3> <ul class="type"> <li><code>Opts = [opt()]</code></li> <li><code>opt() = {db_dir, string()} | ...</code></li> </ul> <div class="description"> <p>Starts a supervisor for the SNMP agent system without a master agent. The supervisor starts all involved SNMP processes, but no agent processes. Sub-agents should be started by calling <code>start_sub_agent/3</code>. </p> <p><code>db_dir</code> is mandatory.</p> <p>See <a href="snmp_config.html#configuration_params" class="seealso">configuration parameters</a> for a description of the options.</p> </div> </div> <div class="function"> <h3 id="start_master_sup/1">start_master_sup(Opts) -&gt; {ok, pid()} | {error, {already_started, pid()}} | {error, Reason}</h3> <ul class="type"> <li><code>Opts = [opt()]</code></li> <li><code>opt() = {db_dir, string()} | {config, ConfOpts()} | ...</code></li> <li><code>ConfOpts = [conf_opts()]</code></li> <li><code>conf_opts() = {dir, string()} | ...</code></li> <li><code>Reason = term()</code></li> </ul> <div class="description"> <p>Starts a supervisor for the SNMP agent system.
