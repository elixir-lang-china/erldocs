<h1>ssl_session_cache_api</h1> <h2 class="modsummary">Defines the API for the TLS session cache so that the data storage scheme can be replaced by defining a new callback module implementing this API.</h2> <div class="section"> <h4>Common Data Types</h4> <p>The following data types are used in the functions below: </p> <p><code>cache_ref() = opaque()</code></p> <p><code>key() = {partialkey(), session_id()}</code></p> <p><code>partialkey() = opaque()</code></p> <p><code>session_id() = binary()</code></p> <p><code>session() = opaque()</code></p> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="delete/2">delete(Cache, Key) -&gt; _</h3> <ul class="type"> <li><code> Cache = cache_ref()</code></li> <li><code> Key = key()</code></li> </ul> <div class="description"> <p> Deletes a cache entry. Will only be called from the cache handling process. </p> </div> </div> <div class="function"> <h3 id="foldl/3">foldl(Fun, Acc0, Cache) -&gt; Acc</h3> <ul class="type"> <li> </li> </ul> <div class="description"> <p>Calls Fun(Elem, AccIn) on successive elements of the cache, starting with AccIn == Acc0. Fun/2 must return a new accumulator which is passed to the next call. The function returns the final value of the accumulator. Acc0 is returned if the cache is empty. </p> </div> </div> <div class="function"> <h3 id="init/0">init() -&gt; opaque() </h3> <ul class="type"> <li> </li> </ul> <div class="description"> <p>Performs possible initializations of the cache and returns a reference to it that will be used as parameter to the other api functions. Will be called by the cache handling processes init function, hence putting the same requirements on it as a normal process init function. </p> </div> </div> <div class="function"> <h3 id="lookup/2">lookup(Cache, Key) -&gt; Entry</h3> <ul class="type"> <li><code> Cache = cache_ref()</code></li> <li><code> Key = key()</code></li> <li><code> Entry = session() | undefined </code></li> </ul> <div class="description"> <p>Looks up a cache entry. Should be callable from any process. </p> </div> </div> <div class="function"> <h3 id="select_session/2">select_session(Cache, PartialKey) -&gt; [session()]</h3> <ul class="type"> <li><code> Cache = cache_ref()</code></li> <li><code> PartialKey = partialkey()</code></li> <li><code> Session = session()</code></li> </ul> <div class="description"> <p>Selects sessions that could be reused. Should be callable from any process. </p> </div> </div> <div class="function"> <h3 id="terminate/1">terminate(Cache) -&gt; _</h3> <ul class="type"> <li><code>Cache = term() - as returned by init/0</code></li> </ul> <div class="description"> <p>Takes care of possible cleanup that is needed when the cache handling process terminates. </p> </div> </div> <div class="function"> <h3 id="update/3">update(Cache, Key, Session) -&gt; _</h3> <ul class="type"> <li><code> Cache = cache_ref()</code></li> <li><code> Key = key()</code></li> <li><code> Session = session()</code></li> </ul> <div class="description"> <p> Caches a new session or updates a already cached one. Will only be called from the cache handling process. </p> </div> </div> </div>
