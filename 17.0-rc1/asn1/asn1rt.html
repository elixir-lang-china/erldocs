<h1>asn1rt</h1> <h2 class="modsummary">ASN.1 runtime support functions</h2> <div class="description"> <p>This module is the interface module for the ASN.1 runtime support functions. To encode and decode ASN.1 types in runtime the functions in this module should be used.</p> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="decode/3">decode(Module,Type,Bytes) -&gt; {ok,Value}|{error,Reason}</h3> <ul class="type"> <li><code>Module = Type = atom()</code></li> <li><code>Value = Reason = term()</code></li> <li><code>Bytes = binary</code></li> </ul> <div class="description"> <p>Decodes <code>Type</code> from <code>Module</code> from the binary <code>Bytes</code>. Returns <code>{ok,Value}</code> if successful.</p> </div> </div> <div class="function"> <h3 id="encode/3">encode(Module,Type,Value)-&gt; {ok,Bytes} | {error,Reason}</h3> <ul class="type"> <li><code>Module = Type = atom()</code></li> <li><code>Value = term()</code></li> <li><code>Bytes = binary</code></li> <li><code>Reason = term()</code></li> </ul> <div class="description"> <p>Encodes <code>Value</code> of <code>Type</code> defined in the ASN.1 module <code>Module</code>. Returns a possibly nested list of bytes and or binaries if successful. To get as fast execution as possible the encode function only performs rudimentary tests that the input <code>Value</code> is a correct instance of <code>Type</code>. The length of strings is for example not always checked. </p> <div class="note"><h2>Note!</h2> <p>Starting in R16, <code>Bytes</code> is always a binary.</p> </div> </div> </div> <div class="function"> <h3 id="info/1">info(Module) -&gt; {ok,Info} | {error,Reason}</h3> <ul class="type"> <li><code>Module = atom()</code></li> <li><code>Info = list()</code></li> <li><code>Reason = term()</code></li> </ul> <div class="description"> <p><code>info/1</code> returns the version of the asn1 compiler that was used to compile the module. It also returns the compiler options that was used.</p> </div> </div> <div class="function"> <h3 id="utf8_binary_to_list/1">utf8_binary_to_list(UTF8Binary) -&gt; {ok,UnicodeList} | {error,Reason}</h3> <ul class="type"> <li><code>UTF8Binary = binary()</code></li> <li><code>UnicodeList = [integer()]</code></li> <li><code>Reason = term()</code></li> </ul> <div class="description"> <p><code>utf8_binary_to_list/1</code> Transforms a UTF8 encoded binary to a list of integers, where each integer represents one character as its unicode value. The function fails if the binary is not a properly encoded UTF8 string.</p> </div> </div> <div class="function"> <h3 id="utf8_list_to_binary/1">utf8_list_to_binary(UnicodeList) -&gt; {ok,UTF8Binary} | {error,Reason}</h3> <ul class="type"> <li><code>UnicodeList = [integer()]</code></li> <li><code>UTF8Binary = binary()</code></li> <li><code>Reason = term()</code></li> </ul> <div class="description"> <p><code>utf8_list_to_binary/1</code> Transforms a list of integers, where each integer represents one character as its unicode value, to a UTF8 encoded binary.</p> </div> </div> </div>
