<h1>httpd_socket</h1> <h2 class="modsummary">Communication utility functions to be used by the Erlang Web server API programmer.</h2> <div class="description"> <p>This module provides the Erlang Web server API module programmer with utility functions for generic sockets communication. The appropriate communication mechanism is transparently used, that is <code>ip_comm</code> or <code>ssl</code>.</p> <span id="deliver"> </span> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="deliver/3">deliver(SocketType, Socket, Data) -&gt; Result</h3> <ul class="type"> <li><code>SocketType = socket_type()</code></li> <li><code>Socket = socket()</code></li> <li><code>Data = io_list() | binary()</code></li> <li><code>Result = socket_closed | void()</code></li> </ul> <div class="description"> <span id="deliver"> </span> <p><code>deliver/3</code> sends the <code>Binary</code> over the <code>Socket</code> using the specified <code>SocketType</code>. Socket and SocketType should be the socket and the socket_type form the mod record as defined in httpd.hrl</p> <span id="peername"> </span> </div> </div> <div class="function"> <h3 id="peername/2">peername(SocketType,Socket) -&gt; {Port,IPAddress}</h3> <ul class="type"> <li><code>SocketType = socket_type()</code></li> <li><code>Socket = socket()</code></li> <li><code>Port = integer()</code></li> <li><code>IPAddress = string()</code></li> </ul> <div class="description"> <span id="peername"> </span> <p><code>peername/3</code> returns the <code>Port</code> and <code>IPAddress</code> of the remote <code>Socket</code>. </p> <span id="resolve"> </span> </div> </div> <div class="function"> <h3 id="resolve/0">resolve() -&gt; HostName</h3> <ul class="type"> <li><code>HostName = string()</code></li> </ul> <div class="description"> <span id="resolve"> </span> <p><code>resolve/0</code> returns the official <code>HostName</code> of the current host. </p> </div> </div> </div> <div class="section"> <span id="see_also"> </span> <h4>SEE ALSO</h4> <p><a href="httpd.html" class="seealso">httpd(3)</a></p> </div>
