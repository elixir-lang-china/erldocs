<h1>tags</h1> <h2 class="modsummary">Generate Emacs TAGS file from Erlang source files</h2> <div class="description"> <p>A <code>TAGS</code> file is used by Emacs to find function and variable definitions in any source file in large projects. This module can generate a <code>TAGS</code> file from Erlang source files. It recognises functions, records, and macro definitions.</p> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="file/2">file(File [, Options])</h3> <div class="description"> <p>Create a <code>TAGS</code> file for the file <code>File</code>.</p> </div> </div> <div class="function"> <h3 id="files/2">files(FileList [, Options])</h3> <div class="description"> <p>Create a TAGS file for the files in the list <code>FileList</code>.</p> </div> </div> <div class="function"> <h3 id="dir/2">dir(Dir [, Options])</h3> <div class="description"> <p>Create a TAGS file for all files in directory <code>Dir</code>.</p> </div> </div> <div class="function"> <h3 id="dirs/2">dirs(DirList [, Options])</h3> <div class="description"> <p>Create a TAGS file for all files in any directory in <code>DirList</code>.</p> </div> </div> <div class="function"> <h3 id="subdir/2">subdir(Dir [, Options])</h3> <div class="description"> <p>Descend recursively down the directory <code>Dir</code> and create a <code>TAGS</code> file based on all files found.</p> </div> </div> <div class="function"> <h3 id="subdirs/2">subdirs(DirList [, Options])</h3> <div class="description"> <p>Descend recursively down all the directories in <code>DirList</code> and create a <code>TAGS</code> file based on all files found.</p> </div> </div> <div class="function"> <h3 id="root/1">root([Options])</h3> <div class="description"> <p>Create a <code>TAGS</code> file covering all files in the Erlang distribution.</p> </div> </div> </div> <div class="section"> <h4>OPTIONS</h4> <p>The functions above have an optional argument, <code>Options</code>. It is a list which can contain the following elements:</p> <list type="bulleted"> <item><code>{outfile, NameOfTAGSFile}</code> Create a <code>TAGS</code> file named <code>NameOfTAGSFile</code>. </item> <item><code>{outdir, NameOfDirectory}</code> Create a file named <code>TAGS</code> in the directory <code>NameOfDirectory</code>.</item> </list> <p>The default behaviour is to create a file named <code>TAGS</code> in the current directory.</p> </div> <div class="section"> <h4>Examples</h4> <list type="bulleted"> <item> <p><code>tags:root([{outfile, "root.TAGS"}]).</code> <br /> </p> <p>This command will create a file named <code>root.TAGS</code> in the current directory. The file will contain references to all Erlang source files in the Erlang distribution.</p> </item> <item> <p><code>tags:files(["foo.erl", "bar.erl", "baz.erl"], [{outdir, "../projectdir"}]). </code> <br /> </p> <p>Here we create file named <code>TAGS</code> placed it in the directory <code>../projectdir</code>. The file contains information about the functions, records, and macro definitions of the three files.</p> </item> </list> </div> <div class="section"> <h4>SEE ALSO</h4> <list type="bulleted"> <item>Richard M. Stallman. GNU Emacs Manual, chapter "Editing Programs", section "Tag Tables". Free Software Foundation, 1995. </item> <item>Anders Lindgren. The Erlang editing mode for Emacs. Ericsson, 1998.</item> </list> </div>
