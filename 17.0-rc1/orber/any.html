<h1>any</h1> <h2 class="modsummary">the corba any type</h2> <div class="description"> <p>This module contains functions that gives an interface to the CORBA any type.</p> <p>Note that the <code>any</code> interface in orber does not contain a destroy function because the any type is represented as an Erlang record and therefor will be removed by the garbage collector when not in use.</p> <p>The type <code>TC</code> used below describes an IDL type and is a tuple according to the to the Erlang language mapping.</p> <p>The type <code>Any</code> used below is defined as:</p> <pre class="sh_erlang"> -record(any, {typecode, value}). </pre> <p>where <code>typecode</code> is a TC tuple and <code>value</code> is an Erlang term of the type defined by the typecode field.</p> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="create/0">create() -&gt;
