<h1>ssh_sftp</h1> <h2 class="modsummary">SFTP client.</h2> <div class="description"> <p>This module implements an SFTP (SSH FTP) client. SFTP is a secure, encrypted file transfer service available for SSH.</p> </div> <div class="section"> <h4>DATA TYPES </h4> <p>Type definitions that are used more than once in this module and/or abstractions to indicate the intended use of the data type: </p> <p><code>ssh_connection_ref() - opaque to the user returned by ssh:connect/3</code></p> <p><code>timeout() = infinity | integer() - in milliseconds.</code></p> </div> <div class="section"> <h4>TIMEOUTS </h4> <p>If the request functions for the SFTP channel return {error, timeout} it does not guarantee that the request did not reach the server and was not performed, it only means that we did not receive an answer from the server within the time that was expected.</p> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="start_channel/1">start_channel(ConnectionRef) -&gt; </h3> <h3 id="start_channel/2">start_channel(ConnectionRef, Options) -&gt; </h3> <h3 id="start_channel/2-1">start_channel(Host, Options) -&gt;</h3> <h3 id="start_channel/3">start_channel(Host, Port, Options) -&gt; {ok, Pid} | {ok, Pid, ConnectionRef} | {error, Reason}</h3> <ul class="type"> <li><code>Host = string()</code></li> <li><code>ConnectionRef = ssh_connection_ref()</code></li> <li><code>Port = integer()</code></li> <li><code>Options = [{Option, Value}]</code></li> <li><code>Reason = term()</code></li> </ul> <div class="description"> <p>If no connection reference is provided, a connection is set up and the new connection is returned. The returned pid for this process should be used as input to all other API functions in this module.</p> <p>Options are:</p> <taglist> <dt><code>{timeout, timeout()}</code></dt> <item> <p>The timeout is passed to the ssh_channel start function, and defaults to infinity.</p> </item> </taglist> <p>All other options are directly passed to <a href="ssh.html" class="seealso">ssh:connect/3</a> or ignored if a connection is already provided. </p> </div> </div> <div class="function"> <h3 id="stop_channel/1">stop_channel(ChannelPid) -&gt; ok</h3> <ul class="type"> <li><code>ChannelPid = pid()</code></li> </ul> <div class="description"> <p>Stops an SFTP channel.
