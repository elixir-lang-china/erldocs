<h1>string</h1> <h2 class="modsummary">String Processing Functions</h2> <div class="description"> <p>This module contains functions for string processing.</p> </div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="len/1">len/1</h3> <div class="description"> <p>Returns the number of characters in the string.</p> </div> </div> <div class="function"> <h3 id="equal/2">equal/2</h3> <div class="description"> <p>Tests whether two strings are equal. Returns <code>true</code> if they are, otherwise <code>false</code>.</p> </div> </div> <div class="function"> <h3 id="concat/2">concat/2</h3> <div class="description"> <p>Concatenates two strings to form a new string. Returns the new string.</p> </div> </div> <div class="function"> <h3 id="chr/2">chr/2</h3> <h3 id="rchr/2">rchr/2</h3> <div class="description"> <p>Returns the index of the first/last occurrence of <code><anno>Character</anno></code> in <code><anno>String</anno></code>. <code>0</code> is returned if <code><anno>Character</anno></code> does not occur.</p> </div> </div> <div class="function"> <h3 id="str/2">str/2</h3> <h3 id="rstr/2">rstr/2</h3> <div class="description"> <p>Returns the position where the first/last occurrence of <code><anno>SubString</anno></code> begins in <code><anno>String</anno></code>. <code>0</code> is returned if <code><anno>SubString</anno></code> does not exist in <code><anno>String</anno></code>. For example:</p> <pre class="sh_erlang"> &gt; string:str(" Hello Hello World World ", "Hello World"). 8 </pre> </div> </div> <div class="function"> <h3 id="span/2">span/2</h3> <h3 id="cspan/2">cspan/2</h3> <div class="description"> <p>Returns the length of the maximum initial segment of <code><anno>String</anno></code>, which consists entirely of characters from (not from) <code><anno>Chars</anno></code>.</p> <p>For example:</p> <pre class="sh_erlang"> &gt; string:span("\t abcdef", " \t"). 5 &gt; string:cspan("\t abcdef", " \t"). 0 </pre> </div> </div> <div class="function"> <h3 id="substr/2">substr/2</h3> <h3 id="substr/3">substr/3</h3> <div class="description"> <p>Returns a substring of <code><anno>String</anno></code>, starting at the position <code><anno>Start</anno></code>, and ending at the end of the string or at length <code><anno>Length</anno></code>.</p> <p>For example:</p> <pre class="sh_erlang"> &gt; substr("Hello World", 4, 5). "lo Wo" </pre> </div> </div> <div class="function"> <h3 id="tokens/2">tokens/2</h3> <div class="description"> <p>Returns a list of tokens in <code><anno>String</anno></code>, separated by the characters in <code><anno>SeparatorList</anno></code>.</p> <p>For example:</p> <pre class="sh_erlang"> &gt; tokens("abc defxxghix jkl", "x "). ["abc", "def", "ghi", "jkl"] </pre> </div> </div> <div class="function"> <h3 id="join/2">join/2</h3> <div class="description"> <p>Returns a string with the elements of <code><anno>StringList</anno></code> separated by the string in <code><anno>Separator</anno></code>.</p> <p>For example:</p> <pre class="sh_erlang"> &gt; join(["one", "two", "three"], ", "). "one, two, three" </pre> </div> </div> <div class="function"> <h3 id="chars/2">chars/2</h3> <h3 id="chars/3">chars/3</h3> <div class="description"> <p>Returns a string consisting of <code><anno>Number</anno></code> of characters <code><anno>Character</anno></code>. Optionally, the string can end with the string <code><anno>Tail</anno></code>.</p> </div> </div> <div class="function"> <h3 id="copies/2">copies/2</h3> <div class="description"> <p>Returns a string containing <code><anno>String</anno></code> repeated <code><anno>Number</anno></code> times.</p> </div> </div> <div class="function"> <h3 id="words/1">words/1</h3> <h3 id="words/2">words/2</h3> <div class="description"> <p>Returns the number of words in <code><anno>String</anno></code>, separated by blanks or <code><anno>Character</anno></code>.</p> <p>For example:</p> <pre class="sh_erlang"> &gt; words(" Hello old boy!", $o). 4 </pre> </div> </div> <div class="function"> <h3 id="sub_word/2">sub_word/2</h3> <h3 id="sub_word/3">sub_word/3</h3> <div class="description"> <p>Returns the word in position <code><anno>Number</anno></code> of <code><anno>String</anno></code>. Words are separated by blanks or <code><anno>Character</anno></code>s.</p> <p>For example:</p> <pre class="sh_erlang"> &gt; string:sub_word(" Hello old boy !",3,$o). "ld b" </pre> </div> </div> <div class="function"> <h3 id="strip/1">strip/1</h3> <h3 id="strip/2">strip/2</h3> <h3 id="strip/3">strip/3</h3> <div class="description"> <p>Returns a string, where leading and/or trailing blanks or a number of <code><anno>Character</anno></code> have been removed. <code><anno>Direction</anno></code> can be <code>left</code>, <code>right</code>, or <code>both</code> and indicates from which direction blanks are to be removed. The function <code>strip/1</code> is equivalent to <code>strip(String, both)</code>.</p> <p>For example:</p> <pre class="sh_erlang"> &gt; string:strip("...Hello.....", both, $.). "Hello" </pre> </div> </div> <div class="function"> <h3 id="left/2">left/2</h3> <h3 id="left/3">left/3</h3> <div class="description"> <p>Returns the <code><anno>String</anno></code> with the length adjusted in accordance with <code><anno>Number</anno></code>. The left margin is fixed. If the <code>length(<anno>String</anno>)</code> &lt; <code><anno>Number</anno></code>, <code><anno>String</anno></code> is padded with blanks or <code><anno>Character</anno></code>s.</p> <p>For example:</p> <pre class="sh_erlang"> &gt; string:left("Hello",10,$.). "Hello....." </pre> </div> </div> <div class="function"> <h3 id="right/2">right/2</h3> <h3 id="right/3">right/3</h3> <div class="description"> <p>Returns the <code><anno>String</anno></code> with the length adjusted in accordance with <code><anno>Number</anno></code>. The right margin is fixed. If the length of <code>(<anno>String</anno>)</code> &lt; <code><anno>Number</anno></code>, <code><anno>String</anno></code> is padded with blanks or <code><anno>Character</anno></code>s.</p> <p>For example:</p> <pre class="sh_erlang"> &gt; string:right("Hello", 10, $.). ".....Hello" </pre> </div> </div> <div class="function"> <h3 id="centre/2">centre/2</h3> <h3 id="centre/3">centre/3</h3> <div class="description"> <p>Returns a string, where <code><anno>String</anno></code> is centred in the string and surrounded by blanks or characters. The resulting string will have the length <code><anno>Number</anno></code>.</p> </div> </div> <div class="function"> <h3 id="sub_string/2">sub_string/2</h3> <h3 id="sub_string/3">sub_string/3</h3> <div class="description"> <p>Returns a substring of <code><anno>String</anno></code>, starting at the position <code><anno>Start</anno></code> to the end of the string, or to and including the <code><anno>Stop</anno></code> position.</p> <p>For example:</p> <pre class="sh_erlang"> sub_string("Hello World", 4, 8). "lo Wo" </pre> </div> </div> <div class="function"> <h3 id="to_float/1">to_float/1</h3> <div class="description"> <p>Argument <code><anno>String</anno></code> is expected to start with a valid text represented float (the digits being ASCII values). Remaining characters in the string after the float are returned in <code><anno>Rest</anno></code>.</p> <p>Example:</p> <pre class="sh_erlang"> &gt; {F1,Fs} = string:to_float("1.0-1.0e-1"), &gt; {F2,[]} = string:to_float(Fs), &gt; F1+F2. 0.9 &gt; string:to_float("3/2=1.5"). {error,no_float} &gt; string:to_float("-1.5eX"). {-1.5,"eX"}</pre> </div> </div> <div class="function"> <h3 id="to_integer/1">to_integer/1</h3> <div class="description"> <p>Argument <code><anno>String</anno></code> is expected to start with a valid text represented integer (the digits being ASCII values). Remaining characters in the string after the integer are returned in <code><anno>Rest</anno></code>.</p> <p>Example:</p> <pre class="sh_erlang"> &gt; {I1,Is} = string:to_integer("33+22"), &gt; {I2,[]} = string:to_integer(Is), &gt; I1-I2. 11 &gt; string:to_integer("0.5"). {0,".5"} &gt; string:to_integer("x=2"). {error,no_integer}</pre> </div> </div> <div class="function"> <name name="to_lower" arity="1" clause_i="1" /> <name name="to_lower" arity="1" clause_i="2" /> <name name="to_upper" arity="1" clause_i="1" /> <name name="to_upper" arity="1" clause_i="2" /> <type variable="String" name_i="1" /> <type variable="Result" name_i="1" /> <type variable="Char" /> <type variable="CharResult" /> <div class="description"> <p>The given string or character is case-converted. Note that the supported character set is ISO/IEC 8859-1 (a.k.a. Latin 1), all values outside this set is unchanged</p> </div> </div> </div> <div class="section"> <h4>Notes</h4> <p>Some of the general string functions may seem to overlap each other. The reason for this is that this string package is the combination of two earlier packages and all the functions of both packages have been retained. </p> <div class="note"><h2>Note!</h2> <p>Any undocumented functions in <code>string</code> should not be used.</p> </div> </div>
