<h1>edoc_layout</h1> <h2 class="modsummary">The standard HTML layout module for EDoc.</h2> <div class="description"> <p>The standard HTML layout module for EDoc. See the <a href="edoc.html" class="seealso">edoc</a> module for details on usage.</p></div> <div class="functions"><h4>Functions</h4><hr /> <div class="function"> <h3 id="module/2">module(Element, Options) -&gt; term() </h3> <div class="description"><span id="module-2"> </span> <p>The layout function.</p> <p>Options to the standard layout: </p><taglist> <dt><code>{index_columns, integer()}</code> </dt> <item><p>Specifies the number of column pairs used for the function index tables. The default value is 1. </p></item> <dt><code>{pretty_printer, atom()}</code> </dt> <item><p>Specifies how types and specifications are pretty printed. If the value <code>erl_pp</code> is specified the Erlang pretty printer (the module <code>erl_pp</code>) will be used. The default is to do no pretty printing which implies that lines can be very long. </p></item> <dt><code>{stylesheet, string()}</code> </dt> <item><p>Specifies the URI used for referencing the stylesheet. The default value is <code>"stylesheet.css"</code>. If an empty string is specified, no stylesheet reference will be generated. </p></item> <dt><code>{sort_functions, boolean()}</code> </dt> <item><p>If <code>true</code>, the detailed function descriptions are listed by name, otherwise they are listed in the order of occurrence in the source file. The default value is <code>true</code>. </p></item> <dt><code>{xml_export, Module::atom()}</code> </dt> <item><p>Specifies an <url href="http://www.erlang.org/edoc/doc/xmerl/doc/index.html">xmerl</url> callback module to be used for exporting the documentation. See <url href="http://www.erlang.org/edoc/doc/xmerl/doc/xmerl.xml#export_simple-3">//xmerl/xmerl:export_simple/3</url> for details. </p></item> </taglist><p> </p> <p><em>See also:</em> <a href="edoc.html#layout-2" class="seealso">edoc:layout/2</a>.</p> </div></div> <div class="function"> <h3 id="type/1">type(E) -&gt; term() </h3> <div class="description"><span id="type-1"> </span> </div></div> <div class="function"> <h3 id="package/2">package(E, Options) -&gt; term() </h3> <div class="description"><span id="package-2"> </span> </div></div> <div class="function"> <h3 id="overview/2">overview(E, Options) -&gt; term() </h3> <div class="description"><span id="overview-2"> </span> </div></div></div> <div class="section"><h4>See also</h4><p><a href="edoc.html" class="seealso">edoc</a></p></div> <authors> <aname>Richard Carlsson</aname> <email>carlsson.richard@gmail.com</email></authors>
