The default value is 20 seconds.</item> <item><code>{filterOp, OperationType}</code> - determine which type of Administrator objects should be started, i.e., <code>'OR_OP'</code> or <code>'AND_OP'</code>. The default value is <code>'OR_OP'</code>.</item> <item><code>{timeService, TimeServiceObj | 'undefined'}</code> - to be able to use Start and/or Stop QoS this option must be used. The default value is <code>'undefined'</code>.</item> <item><code>{filterOp, OperationType}</code> - determine which type of Administrator objects should be started, i.e., <code>'OR_OP'</code> or <code>'AND_OP'</code>. The default value is <code>'OR_OP'</code>.</item> <item><code>{gcTime, Seconds}</code> - this option determines how often, for example, proxies will garbage collect expired events. The default value is 60.</item> <item><code>{gcLimit, Amount}</code> - determines how many events will be stored before, for example, proxies will garbage collect expired events. The default value is 50.
